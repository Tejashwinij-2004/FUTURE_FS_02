# FUTURE_FS_02

Client Lead Management System (Mini CRM) developed as part of Full Stack Web Development Internship. This project allows users to add, view, update, and manage client leads using Node.js, Express, HTML, CSS, and JavaScript.

## Features
- Add new leads
- View all leads
- Update lead status (new / contacted / converted)
- Add notes
- Delete leads

## Technologies Used
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express

## How to Run
1. npm install
2. node server.js
3. Open http://localhost:5000

## Description
This application helps manage client leads efficiently using CRUD operations.
